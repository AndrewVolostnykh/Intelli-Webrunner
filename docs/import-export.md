# Import and Export

## Tree menu
Right-click in the tree to access:
1. `Import .http`
2. `Export .http`
3. `Import OpenAPI`
4. `Export OpenAPI`

## `ƒ<r` menu
The `ƒ<r` button opens:
1. `Import Collections (JSON)`
2. `Export Collections (JSON)`
3. `Import .http`
4. `Export .http`

## Formats
1. `.http` is the IntelliJ HTTP file format.
2. `OpenAPI` imports and exports OpenAPI files.
3. `Collections (JSON)` is the internal Intelli Webrunner collections format.
