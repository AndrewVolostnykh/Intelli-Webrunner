# Відповідь і логи

Після виконання запиту оновлюється нижня панель.

## Response Body
1. Для HTTP показується тіло відповіді.
2. Для gRPC показується тіло відповіді і статус у заголовку.

## Response Headers
1. Headers показуються у форматі JSON.

## Logs
1. Логи включають повідомлення з `Before Request` і `After Request`.
2. Логи також зʼявляються у Debug Call.
