# Імпорт та експорт

## Меню дерева
Контекстне меню дерева (правий клік) містить:
1. `Import .http`
2. `Export .http`
3. `Import OpenAPI`
4. `Export OpenAPI`

## Меню `ƒ<r`
Кнопка `ƒ<r` відкриває:
1. `Import Collections (JSON)`
2. `Export Collections (JSON)`
3. `Import .http`
4. `Export .http`

## Формати
1. `.http` — формат IntelliJ HTTP.
2. `OpenAPI` — імпорт та експорт OpenAPI файлів.
3. `Collections (JSON)` — внутрішній формат колекцій Intelli Webrunner.
