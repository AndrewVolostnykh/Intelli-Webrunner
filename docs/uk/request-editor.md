# Headers, Params, Body

Редактор запиту має вкладки:
1. `Body`
2. `Params`
3. `Headers`
4. `Before Request`
5. `After Request`

## Body
1. `Raw` показує текстовий редактор.
2. `Form Data` показує таблицю `name`, `value`, `type`, `enabled`.
3. `Binary` дозволяє вибрати файл для відправки як байти.

## Params
1. Params додаються до URL при виконанні.
2. Params підлягають шаблонізації.

## Headers
1. Headers задаються таблицею `name`, `value`, `enabled`.
2. Вимкнені рядки не відправляються.
3. Headers підлягають шаблонізації.

## Auto-save
Усі поля зберігаються автоматично під час редагування.
