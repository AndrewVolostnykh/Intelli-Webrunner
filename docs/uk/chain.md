# Chain mode

Chain mode виконує список запитів послідовно зі спільним `vars`.

## Основні дії
1. Додавайте запити у список.
2. Перетягуйте для зміни порядку.
3. `Run` виконує всі запити.
4. `Debug` вмикає покроковий режим.
5. `Next` запускає наступний запит у debug режимі.

## Спільний стан
1. `vars` спільний для всіх кроків.
2. Логи накопичуються у Chain logs.
3. Поточний стан показується у Chain state.
