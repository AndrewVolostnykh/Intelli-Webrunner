# Response Viewer

After execution, the bottom panel updates.

## Response Body
1. HTTP shows the response body.
2. gRPC shows the response body and a status line.

## Response Headers
1. Headers are shown as JSON.

## Logs
1. Logs include messages from `Before Request` and `After Request`.
2. Debug Call stages also print logs.
