# Quickstart

This section helps you create and run your first request.

1. Open the Intelli Webrunner tool window.
2. Create a folder or a request in the left tree.
3. For HTTP:
4. Select method and payload type.
5. Enter URL.
6. Fill Body, Headers, Params if needed.
7. Click `Send`.
8. For gRPC:
9. Enter Target and click `Reload`.
10. Select Service and Method.
11. Fill Body and Headers.
12. Click `Send`.

The response and logs appear in the bottom panel.
